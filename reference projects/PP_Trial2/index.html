Help

This form is currently in locked (user) mode. You can view all information but only edit specific fields.

Available Widgets

  • Tools — Milling Calculator, Drilling Calculator, Metric Converter, and Weight Calculator for quick shop-floor calculations.
  • Request Assistance — Send a request to a programmer for help.
  • Form Feedback — Submit feedback or suggestions about the form.
  • Quick Links — Handy links to external resources like FSWizard.

Expanding Operations

Click the ▼ arrow next to any Primary Operation or Secondary Process header to expand or collapse it. Only one operation can be expanded at a time.

Machine Coordinates

Inside each operation you will find Machine Coord fields for X, Y, Z, and W axes. Type your machine coordinates directly into these fields — they remain editable even when the form is locked.

Note: This function is for visibility only. It replicates the old process of adding a text box to a PDF process packet to add machine coordinates. These do not save. Your coordinates must still be entered into Proshop.

Tool Sign-Off

In the tooling table of each operation, find the Initials column. Type your initials into the cell next to each tool to sign off that it has been verified or set up.

Light Mode

Click the sun icon in the header toolbar (next to the lock button) to toggle between dark and light mode. Light mode switches the background from black to white and inverts text colors for easier viewing in bright environments. This is a display preference only — it does not save with the form. Each time you open a form it will default to dark mode.

Why the New Process Packet?

This new process packet replaces the legacy PDF-based system with a more robust platform. It allows us to build stronger tools, integrate directly with shop systems like Proshop and Zoller TMS, and develop programming automation tools — all of which were not possible with the old format.
