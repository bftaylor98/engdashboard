# Documentation Directory

This directory contains all project documentation, including database schemas, guides, examples, and reference materials.

## Contents

### Database Documentation

- **`DATABASE_SCHEMA_DOCUMENTATION.md`** - Complete database schema documentation
- **`DATABASE_SCHEMA.json`** - Machine-readable schema export
- **`DATABASE_QUERY_GUIDE.md`** - Guide for writing database queries
- **`SQL_PROBE_TOOL_DOCUMENTATION.md`** - Documentation for the sql_probe utility

### Component Documentation

- **`COMPONENT_C-112_COMPLETE_INFO.md`** - Detailed example for component C-112
- **`COMPONENT_IDS_DOCUMENTATION.md`** - Component ID system documentation
- **`component_C-112_full_info.json`** - JSON export of component C-112 data
- **`component_ids.csv`** - CSV export of component IDs
- **`component_ids.json`** - JSON export of component IDs

### Project Documentation

- **`PROJECT_STRUCTURE.md`** - Complete project structure and organization guide
- **`ORGANIZATION_SUMMARY.md`** - Summary of directory reorganization
- **`PART_NUMBER_HYPERLINK_LIBRARY.md`** - Part number hyperlink library documentation
- **`MASTER_DOCUMENTATION_INDEX.md`** - Master index of all documentation

## Quick Reference

### For Database Queries
- Start with `DATABASE_SCHEMA_DOCUMENTATION.md` to understand table structure
- Use `DATABASE_QUERY_GUIDE.md` for query patterns and examples
- Reference `COMPONENT_C-112_COMPLETE_INFO.md` for component query examples

### For Building Reports
- See `PART_NUMBER_HYPERLINK_LIBRARY.md` for hyperlink functionality
- Reference `templates/README.md` for report generation patterns

### For Project Structure
- See `PROJECT_STRUCTURE.md` for complete directory organization
- See `ORGANIZATION_SUMMARY.md` for recent changes

## File Formats

- **`.json`** - JSON data exports and schemas
- **`.csv`** - CSV data exports
