# Test Scripts

This directory contains test and utility scripts for database queries and component testing.

## Scripts

### `test_component_c212.py`
Test script to query ComponentID C-212 and display:
- Component basic information (ID, code, description)
- Part number (OrderNo)
- UnitPrice
- Transaction history
- Price information

**Usage:**
```bash
python tests/test_component_c212.py
```

This script can be modified to test other component IDs by changing the component code in the script.
